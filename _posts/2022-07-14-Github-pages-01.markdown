## 글을 시작하며,
Github Page를 이용하여 블로그를 만들면서 나름대로 찾아본 내용들을 정리하여 공유하고자 한다.

먼저 Github 계정과 해당 블로그를 작업할 저장소(Repositories)가 필요하다. 자신의 Github에서 Repositories 항목을 찾을 수 있는데, 이것이 Git으로 관리하는 프로젝트 저장소를 뜻한다.
